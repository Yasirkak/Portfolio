Description

Schoolbook is a prototype developed by me and my team for schools, ranging from the first grade till the ninth. Here the teachers can post both pictures and relevant information regarding different students. Teachers can even contact parents directly though a chatt function and update different schedules. Easier put, Schoolbook is a socialmedia platform between teachers and parents.

My part

  • Took part in Empathize, Ideate and Define
  • Developed the main page with HTML and CSS
  • Developed the chatt function with HTML, CSS and Javascript
  • Developed the login function with Javascript
